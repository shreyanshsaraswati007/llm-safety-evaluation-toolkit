# LLM Safety Evaluation Toolkit

This project evaluates the vulnerability of large language models (LLMs) to prompt injection and jailbreak attacks.

## Features
- Adversarial prompt dataset (40+ prompts)
- Multiple attack categories:
  - Instruction override
  - Role-play jailbreak
  - Multi-step reasoning
  - Context manipulation
- Defense mechanisms:
  - Input filtering
  - System prompt defense
- Vulnerability classification system
- Visualization of results

## Results
The system measures:
- Vulnerability rate
- Defense effectiveness
- Response safety distribution

## Tech Stack
- Python
- HuggingFace Transformers
- Pandas
- Matplotlib

## Goal
This project aims to explore real-world weaknesses in LLM safety and evaluate lightweight defense strategies.
